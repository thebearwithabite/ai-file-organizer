## 2024-05-24 - File Upload Zone Keyboard Accessibility
**Learning:** For screen reader users interacting with drop zones in React, dynamically changing states (like "Drag & drop a file here" vs. "Drop file here") need `aria-live="polite"` and `aria-atomic="true"` to announce the state changes. Interactive fallback elements (like the "Browse Files" button) need explicit `focus-visible` styles to ensure valid keyboard navigation and a focus ring that matches the rest of the theme. Also decorative icons like `<Upload>` should be hidden with `aria-hidden="true"`.
**Action:** Always wrap dynamically changing content inside a dropzone in `aria-live` containers and apply `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-primary` to all custom buttons within it. Hide decorative SVGs.
